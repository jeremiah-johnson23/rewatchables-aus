## Features

- **Episode Database** - All Rewatchables episodes with movie details, hosts, and Spotify links
- **Australian Streaming** - See which services have each movie (Netflix, Stan, Prime Video, Disney+, Binge, Paramount+, Apple TV+)
- **Search & Filter** - Find episodes by movie title, host, director, or genre
- **Hall of Fame** - Filter to see only the Hall of Fame episodes
- **Community Ratings** - See how the community rates each episode
- **Mobile Friendly** - Works great on phones and tablets

## Contributing

We welcome contributions from the community! Here's how you can help:

### Report Streaming Updates

Streaming availability changes frequently. If you notice a movie is now available (or no longer available) on a service:

1. [Open a Streaming Update issue](https://github.com/jeremiah-johnson23/rewatchables-aus/issues/new?template=streaming-update.md)
2. Fill in the movie details and what changed
3. Verify with [JustWatch AU](https://www.justwatch.com/au) if possible

### Request New Episodes

Want to add an episode we're missing?

1. [Open a New Episode issue](https://github.com/jeremiah-johnson23/rewatchables-aus/issues/new?template=new-episode.md)
2. Include as much detail as you can
3. We'll add it to the database

### Direct Contributions

If you're comfortable with GitHub:

1. Fork the repository
2. Edit `src/data/episodes.json`
3. Submit a pull request

See [CONTRIBUTING.md](CONTRIBUTING.md) for detailed instructions.

## Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/jeremiah-johnson23/rewatchables-aus.git
   cd rewatchables-aus
   ```

2. Start a local server (any of these work):
   ```bash
   # Python
   python -m http.server 8000

   # Node.js (if you have npx)
   npx serve

   # PHP
   php -S localhost:8000
   ```

3. Open http://localhost:8000 in your browser
